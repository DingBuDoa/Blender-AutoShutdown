# Blender 自动关机插件

实现渲染动画后自动关闭电脑  
单帧渲染不触发支持中途取消。

兼容 **Blender 3.6 及以上版本**。

---

## 功能

- 动画渲染完成自动执行系统关机
- 延迟可选：立即 / 1分钟 / 5分钟 / 10分钟 / 30分钟 /1小时
- 随时取消计划好的关机任务
- 调试模式：只弹提示框，不真关机

---

## 安装
1. 下载 `auto_shutdown_v1.0.0.zip`（从 Releases 页面获取）
2. Blender 里打开：`编辑` → `偏好设置` → `插件` → `安装`
3. 选择 zip 文件，勾选启用

如果有旧版，建议先卸载再装。
---

## 使用方法

1. 在插件面板里勾上本插件
2. 下拉菜单选一个延迟时间（默认是“立即”）
3. 正常按 `Ctrl+F12` 渲染动画
4. 渲染完成后，到时间就会自动关机
5. 如果想取消：点一下“取消定时关机”按钮
> 调试模式开启时，不会执行真实关机，可在控制台查看详细,渲染完成后会有报告弹窗
---
## 兼容性

- Blender 3.6以上版本
---

## 一些说明

- 单帧渲染（F12）不会触发
- 延迟关机功能能随时取消
