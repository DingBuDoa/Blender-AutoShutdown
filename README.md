# Blender 自动关机插件

渲染动画（Ctrl+F12）完事后自动关掉电脑。  
单帧渲染不触发，给你留了延迟关机的选项，也能随时取消。

兼容 **Blender 3.6 及以上**。

---

## 功能

- 动画渲染完成自动执行系统关机
- 延迟可选：立即 / 1分钟 / 5分钟 / 10分钟 / 30分钟
- 随时取消计划好的关机任务
- 渲染完成后 **保持勾选状态**，不会自己取消
- 调试模式：只弹提示框，不真关机
- 切换 .blend 文件或开新工程，功能依然有效，不用反复启用
- 系统弹窗 + 控制台日志同时输出

---

## 安装

1. 下载 `auto_shutdown_v1.0.0.zip`（从 Releases 页面获取）
2. Blender 里打开：`编辑` → `偏好设置` → `插件` → `安装`
3. 选择 zip 文件，勾选启用

如果有旧版，建议先卸载再装。

---

## 使用方法

1. 在插件面板里勾上本插件
2. 下拉菜单选一个延迟时间（默认是“立即”）
3. 正常按 `Ctrl+F12` 渲染动画
4. 渲染完成后，到时间就会自动关机
5. 如果想取消：点一下“取消定时关机”按钮

> 调试模式开启时，不会执行真实关机，只弹个窗口告诉你“本来该关机了”。

---

## 兼容性

- Blender 3.6、3.6 LTS、4.0、4.1、4.2 等后续版本（只要不低于 3.6）
- Windows / macOS / Linux（系统关机命令自动适配）

---

## 一些说明

- 单帧渲染（F12）不会触发，放心预览单张
- 延迟关机是为了让你有时间看一眼渲染结果再关机，别误关
- 换工程后插件仍然生效，这是故意设计的，不是 bug
